# Venture Logo Images

Store company logos here for **pixel-perfect quality**.

## Usage

1. Save your logo file here (e.g., `kamioi-logo.png`)
2. In Website Manager, use the path: `/images/ventures/kamioi-logo.png`
3. Logo will render with zero quality loss

## File Format Recommendations

- **PNG**: Best for logos with transparency
- **SVG**: Best for vector logos (scalable, smallest file size)
- **JPG**: Only for photos, not logos

## File Naming

Use lowercase with hyphens:
- ✅ `kamioi-logo.png`
- ✅ `argufight-logo.svg`
- ❌ `KamiOi Logo.PNG`

## File Size

Keep logos under 500KB for fast loading.
